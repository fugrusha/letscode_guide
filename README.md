# Sweater App - simple clone of Tweeter

You can:
- register\login (simple validation)
- post your tweets
- edit your tweets
- pin image to your tweet
- change password, email
- recieve confirmation emails
- subscribe to others
- like/unlike tweets
- search tweets by tag

Using:
- Java
- Spring Boot MVC
- Postgres
- Hibernate
- Flyway
- Bootstrap
- Freemarker
- RECAPTCHA
- Turbolinks
